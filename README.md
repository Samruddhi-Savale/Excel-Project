## Project Overview

This project focuses on analyzing **bike sales data** using **Microsoft Excel** to uncover meaningful business insights and present them through an interactive and easy-to-understand dashboard. The goal is to help stakeholders quickly understand customer behavior, sales trends, and factors influencing bike purchases.

This project demonstrates strong skills in **data cleaning, analysis, and visualization**, making it highly relevant for **Data Analyst / Business Analyst** roles.

---

## Objectives

* Analyze customer demographics and purchasing behavior
* Identify key factors influencing bike sales
* Build an interactive dashboard for quick decision-making
* Present insights in a clear, recruiter-friendly format

---

## Dataset

* The dataset contains customer details, income, age group, gender, commute distance, and bike purchase status.
* Data was cleaned and transformed directly in Excel for analysis.
